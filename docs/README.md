## 🛠️ Development

### Quick Reference
- **Backend**: Python/Flask with PostgreSQL and Kafka
- **Frontend**: React with Leaflet mapping
- **AI**: OpenAI, Claude, and Thaura.ai integration
- **Deployment**: Docker Compose

### Key Components
```
app/
├── __init__.py              # Flask app factory
├── models.py               # Database models
├── summary/                # AI summarization
├── city_history/           # Historical context
└── frontend/src/           # React application
```
